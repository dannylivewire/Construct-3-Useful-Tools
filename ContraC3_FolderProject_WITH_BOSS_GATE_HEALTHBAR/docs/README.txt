OPEN THIS FOLDER AS A PROJECT IN CONSTRUCT 3
--------------------------------------------
1) Construct 3 → Project → Open → Folder project… → pick this folder.
2) Add an Event Sheet called Main, EnemyAI, World.
3) Open eventSheets/*.events.json in a text editor, copy the JSON text, then right-click the sheet in Construct and Paste.
4) Create objects using files/assets images (Player, Enemy, Bullet, Explosion), add behaviors (Platform, Bullet, etc.).
5) Add a Tilemap using tileset.png and a background using bg_parallax.png. Add layers: UI, Main, Mid, Far.
6) Save as .c3p when ready.
